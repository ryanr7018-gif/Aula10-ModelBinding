# Aula 10 - Model Binding

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

## Funcionalidades

- Cadastro de aluno
- Model Binding
- Validação com Data Annotations
- Página de confirmação
- Botão para novo cadastro

## Campos

- Nome
- E-mail
- RA
- Curso
- Data de Nascimento

## Testes

### Positivos
- Cadastro com todos os dados preenchidos corretamente.

### Negativos
- Nome vazio.
- E-mail inválido.
- RA vazio.
- Curso vazio.
- Data de nascimento vazia.
